What stage and availability the app is in

```graphql
enum AppStatus {
  DRAFT
  PRIVATE
  PUBLIC
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>AppStatus.<b>DRAFT</b></code>](#) 
> The app is in the process of being created
> 

#### [<code style={{ fontWeight: 'normal' }}>AppStatus.<b>PRIVATE</b></code>](#) 
> The app is available to install, but only through shared links
> 

#### [<code style={{ fontWeight: 'normal' }}>AppStatus.<b>PUBLIC</b></code>](#) 
> The app is available to install, and is searchable/discoverable
>
