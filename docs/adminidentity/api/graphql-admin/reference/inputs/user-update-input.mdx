Input object to update a User.
Very similiar to User, but does not include sub and email is not required.

```graphql
input UserUpdateInput {
  address: String
  birthdate: String
  email: String
  familyName: String
  gender: String
  givenName: String
  locale: String
  nickname: String
  phone: String
  picture: String
  profile: String
  userName: ID!
  website: String
  zoneinfo: String
}
```
