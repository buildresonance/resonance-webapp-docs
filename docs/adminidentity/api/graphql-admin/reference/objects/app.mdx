Represents a App, which are machine users that perform a small context, unified set of tasks.

```graphql
type App {
  appClientIds: [String!]!
  appId: ID!
  appType: AppType!
  createdAt: String!
  description: String
  name: String!
  partnerId: ID!
  requestedScopes: [String]
  status: AppStatus!
  updatedAt: String!
  updatedBy: ID!
}
```
