Represents a Partner, the parent to an App, and the parent most component for Development

```graphql
type Partner {
  partnerId: ID!
  name: String!
  description: String
  ownerUserName: String!
  createdAt: String!
  updatedAt: String!
  users(
    first: Int = 0
    last: Int = 0
    before: String
    after: String

}
```
