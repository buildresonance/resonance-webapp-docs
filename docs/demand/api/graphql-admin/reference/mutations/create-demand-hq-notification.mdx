Create a new demandHqNotification.
                
Requires the demandhqnotification/write permission on the associated DemandHq.

```graphql
createDemandHqNotification(
  input: DemandHqNotificationCreateInput!
): DemandHqNotification!
```

### Type

#### [`DemandHqNotification`](/demand/api/graphql-admin/reference/objects/demand-hq-notification) <Badge class="secondary" text="object"/>
> An actionable notification for a DemandHq.
A User can acknowledge, ignore, or resolve a notification.
In some cases, a suggested action can be taken by the User to resolve the notification.
