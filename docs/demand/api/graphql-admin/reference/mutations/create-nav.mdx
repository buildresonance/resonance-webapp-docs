Create a new Nav.
                
Requires the "nav/write" permission on the associated DemandHq.

```graphql
createNav(
  input: NavCreateInput!
): Nav
```


### Arguments

#### [`input`](#) ([`NavCreateInput!`](/demand/api/graphql-admin/reference/inputs/nav-create-input))




### Type

#### [`Nav`](/demand/api/graphql-admin/reference/objects/nav)

Nav is an item within a navigational heirarchy of some kind, most often viewed as a menu.
