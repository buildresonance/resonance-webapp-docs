Create a new Product and its associated ContentItems, Locales, Variants, and
also VariantSupplierVariants from an existing SupplierProduct you have access
to. Requires several write permissions and access to a Supplier.

```graphql
createProductCompositeFromSupplierProduct(
  demandHqId: ID!
  supplierId: ID!
  supplierProductId: ID!
): Product!
```
