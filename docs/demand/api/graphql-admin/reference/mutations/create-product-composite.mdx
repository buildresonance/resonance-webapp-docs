Create a new Product and its associated ContentItems, Locales, and Variants in
one mutation. Requires several write permissions and access to a Supplier.

```graphql
createProductComposite(
  duplicationCheckType: CompositeDuplicationCheckType
  input: ProductCompositeCreateInput!
): Product!
```
