Create a new Variant from a Supplier Variant.

Requires variant/write permission on the associated demandHq for the requesting identity.
Additionally, the DemandHq requires permission to the Supplier.

```graphql
createVariantFromSupplierVariant(
  demandHqId: ID!
  productId: ID!
  supplierId: ID!
  supplierVariantId: ID!
): Variant!
```
