Paginate through Collections

```graphql
type CollectionConnectionWithSearch {
  edges: [CollectionEdge]!
  nodes: [Collection]!
  pageInfo: PageInfo!
  querySuggestions: JSON
  termAggregations: CollectionTermAggregations
}
```


### Fields

#### [`edges`](#) ([`[CollectionEdge]!`](/demand/api/graphql-admin/reference/objects/collection-edge))




#### [`nodes`](#) ([`[Collection]!`](/demand/api/graphql-admin/reference/objects/collection))




#### [`pageInfo`](#) ([`PageInfo!`](/demand/api/graphql-admin/reference/objects/page-info))




#### [`querySuggestions`](#) ([`JSON`](/demand/api/graphql-admin/reference/scalars/json))

This is an object with terms in the query as keys, and a suggestion for that
term as the value. This is often referred to as 'Did you mean?'


#### [`termAggregations`](#) ([`CollectionTermAggregations`](/demand/api/graphql-admin/reference/objects/collection-term-aggregations))

Values and the number of times they occur for the results of the query. These
will be the same for every page of a query, and should only be fetched on the first page.
