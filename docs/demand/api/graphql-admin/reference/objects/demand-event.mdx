Contains before and after snapshots for every event that occurs within the Demand domain

```graphql
type DemandEvent {
  message: DemandEventMessage
  messageId: String!
  subject: String
  timestamp: Date!
}
```


### Fields

#### [`message`](#) ([`DemandEventMessage`](/demand/api/graphql-admin/reference/objects/demand-event-message))

This is the event data and metadata.
The data is variable, but most typically contains a newItem and oldItem.


#### [`messageId`](#) ([`String!`](/demand/api/graphql-admin/reference/scalars/string))

UUID


#### [`subject`](#) ([`String`](/demand/api/graphql-admin/reference/scalars/string))

A human-friendly string summarizing the event


#### [`timestamp`](#) ([`Date!`](/demand/api/graphql-admin/reference/scalars/date))

ISO Date that the event occurred
