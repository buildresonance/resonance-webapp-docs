Nav is an item within a navigational heirarchy of some kind, most often viewed as a menu.

```graphql
type Nav {
  createdAt: Date!
  demandHqId: ID!
  href: String!
  label: String!
  navId: ID!
  navLevel: Int!
  navType: String!
  navs(
  after: String
  before: String
  first: Int
  last: Int
): NavConnection!
  parentNavId: ID
  position: Int
  storefrontId: ID!
  updatedAt: Date!
  updatedBy: ID!
}
```
