A discount applied to an OrderItem. Discounts alter pricing of an OrderItem, and may include a coupon code to do so.

```graphql
type OrderItemDiscount {
  code: String
  discountAmount: Float!
  discountType: OrderItemDiscountType!
  discountedAt: Date!
}
```
