This has a many to one relationship with Product, and provides a connection to the CDN items.

```graphql
type ProductContentItem {
  alt: String!
  category: String
  contentItemId: ID!
  contentKey: String!
  contentType: ProductContentType!
  createdAt: Date!
  demandHqId: ID!
  position: Int
  productId: ID!
  updatedAt: Date!
  updatedBy: String!
}
```
