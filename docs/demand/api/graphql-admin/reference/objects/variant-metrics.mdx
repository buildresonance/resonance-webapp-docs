Metrics for a variant, including inventory quantity, status, and supplierVariant counts

```graphql

  demandHqId: ID!
  inventoryQuantity: Int!
  inventoryStatus: InventoryStatus!
  productId: ID!
  updatedAt: String!
  variantId: ID!
}
```
