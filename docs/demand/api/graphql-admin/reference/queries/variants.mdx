Get several variants by either VariantIds or a DemandHqId with Relay pagination.
If VariantIds are provided, all other arguments are ignored.

Requires demandHqvariant/read permission on the requesting identity for all associated demandHqs

```graphql
variants(
  after: String
  before: String
  demandHqId: ID
  first: Int
  last: Int
  variantIds: [ID!]
): VariantConnection!
```
