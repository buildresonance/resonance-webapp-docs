Exposes a URL that specifies the behaviour of this scalar.

```graphql
directive @specifiedBy(
  url: String!
)
```


### Arguments

#### [`url`](#) ([`String!`](/demand/api/graphql-storefront/reference/scalars/string))

The URL that specifies the behaviour of this scalar.
