A customerAddress is an Address entity that is associated with a customer

```graphql
type CustomerAddress {
  city: String!
  company: String
  country: String!
  customerAddressId: ID!
  customerId: ID!
  demandHqId: ID!
  name: String!
  phoneNumber: String
  postalCode: String!
  state: String!
  storefrontId: ID!
  streetAddress1: String!
  streetAddress2: String
}
```
