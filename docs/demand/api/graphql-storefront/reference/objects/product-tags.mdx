These are the tags associated to the product

```graphql
type ProductTags {
  parsed(
  category: String
  key: String
): [ParsedTag]
  raw: [String]
}
```


### Fields

#### [`parsed`](#) ([`[ParsedTag]`](/demand/api/graphql-storefront/reference/objects/parsed-tag))

Get parsed, and optionally filtered, tags for a Product.
- ##### [`category`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Optionally filter tags by category.


- ##### [`key`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Optionally filter tags by key. If category is not provided, but key is, category must be null to be returned.
