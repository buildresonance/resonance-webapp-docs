A Collection is a group of products, what may often be referred to as a "Category Page" of some variety.
In reality, a collection is far more flexible and can be used to represent any
product grouping, and the collectionType can help differentiate between them.

```graphql
type PublicCollection {
  collectionId: ID!
  collectionType: String!
  content: CollectionContent
  demandHqId: ID!
  description: String
  descriptionOrExcerpt(
  numberOfCharacters: Int
): String
  featuredProductIds: [ID]
  primaryImageUrl: String
  products(
  after: String
  before: String
  first: Int
  last: Int
  manufacturer: [StringFilterInput]
  msrp: FloatRangeFilterInput
  numberOfTermAggregations: ProductNumberOfTermAggregations
  price: FloatRangeFilterInput
  query: String
  slugs: [String]
  sortOrder: [ProductSortInput]
  standardizedColor: [ProductStandardizedColorFilterInput]
  tagPrefixes: [StringFilterInput]
  tags: [StringFilterInput]
  vendor: StringFilterInput
): PublicProductConnectionWithSearch!
  publishedAt: Date
  secondaryImageUrl: String
  seoDescription: String
  seoTitle: String
  slug: String!
  subtitle: String
  tags: CollectionTags
  title: String!
}
```


### Fields

#### [`collectionId`](#) ([`ID!`](/demand/api/graphql-storefront/reference/scalars/id))




#### [`collectionType`](#) ([`String!`](/demand/api/graphql-storefront/reference/scalars/string))

Any string to declare a category of this collection, can be used to filter collections, limit 128 characters.


#### [`content`](#) ([`CollectionContent`](/demand/api/graphql-storefront/reference/objects/collection-content))

Get the content of the collection.


#### [`demandHqId`](#) ([`ID!`](/demand/api/graphql-storefront/reference/scalars/id))




#### [`description`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

A short description of the collection, limit 1000 characters.


#### [`descriptionOrExcerpt`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Get the description if it exists, or fallback to the excerpt of content for the collection.
- ##### [`numberOfCharacters`](#) ([`Int`](/demand/api/graphql-storefront/reference/scalars/int))

The number of characters to return, defaults to 200.


#### [`featuredProductIds`](#) ([`[ID]`](/demand/api/graphql-storefront/reference/scalars/id))

This can be used to pin certain products to the top of a collection when using certain sorts.
Max number of products is 20.


#### [`primaryImageUrl`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

A primary image url for this collection, can be generated from a Collection ContentItem.


#### [`products`](#) ([`PublicProductConnectionWithSearch!`](/demand/api/graphql-storefront/reference/objects/public-product-connection-with-search))

Paginate through products in a collection.

Requires the publicproduct/read scope.
- ##### [`after`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Returns the elements in the list that come after the specified cursor.


- ##### [`before`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Returns the elements in the list that come before the specified cursor.


- ##### [`first`](#) ([`Int`](/demand/api/graphql-storefront/reference/scalars/int))

Returns the first n elements from the list.


- ##### [`last`](#) ([`Int`](/demand/api/graphql-storefront/reference/scalars/int))

Returns the last n elements from the list.


- ##### [`manufacturer`](#) ([`[StringFilterInput]`](/demand/api/graphql-storefront/reference/inputs/string-filter-input))

Optionally filter by manufacturer.


- ##### [`msrp`](#) ([`FloatRangeFilterInput`](/demand/api/graphql-storefront/reference/inputs/float-range-filter-input))

Optionally filter by msrp.


- ##### [`numberOfTermAggregations`](#) ([`ProductNumberOfTermAggregations`](/demand/api/graphql-storefront/reference/inputs/product-number-of-term-aggregations))

The number of results to return for each term aggregation. Defaults to 0 for each.


- ##### [`price`](#) ([`FloatRangeFilterInput`](/demand/api/graphql-storefront/reference/inputs/float-range-filter-input))

Optionally filter by price.


- ##### [`query`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Query products for certain terms. Searches several fields, including
product.tags, product.title, product.vendor, productLocales.description,
productLocales.featureBullets, variant.barcode, variant.mpn, variant.sku


- ##### [`slugs`](#) ([`[String]`](/demand/api/graphql-storefront/reference/scalars/string))

Filter Products using slugs


- ##### [`sortOrder`](#) ([`[ProductSortInput]`](/demand/api/graphql-storefront/reference/inputs/product-sort-input))

Sort Products by a series of fields and directions, defaults to CREATED_AT/ASC.


- ##### [`standardizedColor`](#) ([`[ProductStandardizedColorFilterInput]`](/demand/api/graphql-storefront/reference/inputs/product-standardized-color-filter-input))

Optionally filter by standardized color.


- ##### [`tagPrefixes`](#) ([`[StringFilterInput]`](/demand/api/graphql-storefront/reference/inputs/string-filter-input))

Filter Products using Tag Prefixes (starts with).


- ##### [`tags`](#) ([`[StringFilterInput]`](/demand/api/graphql-storefront/reference/inputs/string-filter-input))

Filter Products using Tags.


- ##### [`vendor`](#) ([`StringFilterInput`](/demand/api/graphql-storefront/reference/inputs/string-filter-input))

Optionally filter by vendor.


#### [`publishedAt`](#) ([`Date`](/demand/api/graphql-storefront/reference/scalars/date))

ISO format date when the collection was published. Note that ensuing updates
with status set to PUBLISHED will republish and re-set this date.


#### [`secondaryImageUrl`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

A secondary image url for this collection, often a logo or related item.


#### [`seoDescription`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Description specific for SEO purposes, limit 170 characters.


#### [`seoTitle`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Title specific for SEO purposes, limit 256 characters.


#### [`slug`](#) ([`String!`](/demand/api/graphql-storefront/reference/scalars/string))

Slug for the URL for this collection. Limit 128 characters.


#### [`subtitle`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

The subtitle of the collection, limit 256 characters.


#### [`tags`](#) ([`CollectionTags`](/demand/api/graphql-storefront/reference/objects/collection-tags))

Get the tags of the collection.


#### [`title`](#) ([`String!`](/demand/api/graphql-storefront/reference/scalars/string))

The title of the collection, limit 256 characters.
