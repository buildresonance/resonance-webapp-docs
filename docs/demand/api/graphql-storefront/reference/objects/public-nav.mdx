Nav is an item within a navigational heirarchy of some kind, most often viewed as a menu.

```graphql
type PublicNav {
  demandHqId: ID!
  href: String!
  label: String!
  navId: ID!
  navLevel: Int!
  navType: String!
  navs(
  after: String
  before: String
  first: Int
  last: Int
): PublicNavConnection!
  parentNavId: ID
  position: Int
  storefrontId: ID!
}
```
