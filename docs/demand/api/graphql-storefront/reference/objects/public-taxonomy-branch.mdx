TaxonomyBranch is an item within a taxonomyTree that maps out a Product Taxonomy
hierarchy using Product Tags and Collections.

```graphql
type PublicTaxonomyBranch {
  collectionId: ID
  createdAt: Date!
  demandHqId: ID!
  parentTaxonomyBranchId: ID
  productTag: String!
  storefrontIds: [ID!]
  taxonomyBranchId: ID!
  taxonomyBranches(
  after: String
  before: String
  first: Int
  last: Int
): PublicTaxonomyBranchConnection!
  taxonomyLevel: TaxonomyLevel!
  taxonomyTree: String!
  title: String!
  updatedAt: Date!
  updatedBy: ID!
}
```
