Get all collectionTypeContents for a storefront.
            
Requires the collectiontypecontent/read scope on the associated DemandHq

```graphql
collectionTypeContents(
  after: String
  before: String
  first: Int
  last: Int
): PublicCollectionTypeContentConnection!
```


### Arguments

#### [`after`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Returns the elements in the list that come after the specified cursor.


#### [`before`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Returns the elements in the list that come before the specified cursor.


#### [`first`](#) ([`Int`](/demand/api/graphql-storefront/reference/scalars/int))

Returns the first n elements from the list.


#### [`last`](#) ([`Int`](/demand/api/graphql-storefront/reference/scalars/int))

Returns the last n elements from the list.


### Type

#### [`PublicCollectionTypeContentConnection`](/demand/api/graphql-storefront/reference/objects/public-collection-type-content-connection)

Pagination object for PublicCollectionTypeContent.
