List pages for a Storefront with optional filters.
            
Requires the publicpage/read scope.

```graphql
pages(
  after: String
  author: StringFilterInput
  before: String
  first: Int
  last: Int
  numberOfTermAggregations: PageNumberOfTermAggregations
  pageType: StringFilterInput
  query: String
  slugs: [String]
  sortOrder: [PageSortInput]
  tagPrefixes: [StringFilterInput]
  tags: [StringFilterInput]
): PublicPageConnectionWithSearch!
```


### Arguments

#### [`after`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Returns the elements in the list that come after the specified cursor.


#### [`author`](#) ([`StringFilterInput`](/demand/api/graphql-storefront/reference/inputs/string-filter-input))

Filter Pages using author


#### [`before`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Returns the elements in the list that come before the specified cursor.


#### [`first`](#) ([`Int`](/demand/api/graphql-storefront/reference/scalars/int))

Returns the first n elements from the list.


#### [`last`](#) ([`Int`](/demand/api/graphql-storefront/reference/scalars/int))

Returns the last n elements from the list.


#### [`numberOfTermAggregations`](#) ([`PageNumberOfTermAggregations`](/demand/api/graphql-storefront/reference/inputs/page-number-of-term-aggregations))

The number of results to return for each term aggregation. Defaults to 0 for each.


#### [`pageType`](#) ([`StringFilterInput`](/demand/api/graphql-storefront/reference/inputs/string-filter-input))

Filter Pages using pageType


#### [`query`](#) ([`String`](/demand/api/graphql-storefront/reference/scalars/string))

Query pages for certain terms. Searches several fields, including
page.author, page.description, page.subtitle, page.title, pageTags, and the
actual content of the page.


#### [`slugs`](#) ([`[String]`](/demand/api/graphql-storefront/reference/scalars/string))

Filter Pages using slugs


#### [`sortOrder`](#) ([`[PageSortInput]`](/demand/api/graphql-storefront/reference/inputs/page-sort-input))

Sort Pages by a series of fields and directions.


#### [`tagPrefixes`](#) ([`[StringFilterInput]`](/demand/api/graphql-storefront/reference/inputs/string-filter-input))

Filter Pages using Tag Prefixes (starts with).


#### [`tags`](#) ([`[StringFilterInput]`](/demand/api/graphql-storefront/reference/inputs/string-filter-input))

Filter Pages using Tags.


### Type

#### [`PublicPageConnectionWithSearch`](/demand/api/graphql-storefront/reference/objects/public-page-connection-with-search)

Paginate through PublicPages
