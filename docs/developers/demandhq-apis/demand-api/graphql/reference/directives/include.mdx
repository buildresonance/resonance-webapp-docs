Directs the executor to include this field or fragment only when the `if` argument is true.

```graphql
directive @include(
  if: Boolean!
)
```


### Arguments

#### [`if`](#) ([`Boolean!`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/boolean))

Included when true.
