Create a new CollectionTypeContent.
                
Requires the collectiontypecontent/write scope on the associated DemandHq.

```graphql
createCollectionTypeContent(
  input: CollectionTypeContentCreateInput!
): CollectionTypeContent!
```


### Arguments

#### [`input`](#) ([`CollectionTypeContentCreateInput!`](/developers/demandhq-apis/demand-api/graphql/reference/inputs/collection-type-content-create-input))




### Type

#### [`CollectionTypeContent`](/developers/demandhq-apis/demand-api/graphql/reference/objects/collection-type-content)

This holds content specific to a collectionType. This is generally used on a
page that lists all collections for a collectionType.
