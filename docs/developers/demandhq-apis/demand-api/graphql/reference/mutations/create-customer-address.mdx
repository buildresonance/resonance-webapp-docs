Creates a new CustomerAddress
                
Requires the customeraddress/write scope on the associated DemandHq.


```graphql
createCustomerAddress(
  input: CustomerAddressCreateInput!
): CustomerAddress

```


### Arguments

#### `input` ([`CustomerAddressCreateInput`](/developers/demandhq-apis/demand-api/graphql/reference/inputs/customer-address-create-input))




### Type

#### [`CustomerAddress`](/developers/demandhq-apis/demand-api/graphql/reference/objects/customer-address) 

A customerAddress is an Address entity that is associated with a customer
