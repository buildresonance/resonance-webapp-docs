Creates a new Customer
                
Requires the customer/write scope on the associated DemandHq.


```graphql
createCustomer(
  input: CustomerCreateInput!
): Customer

```


### Arguments

#### `input` ([`CustomerCreateInput`](/developers/demandhq-apis/demand-api/graphql/reference/inputs/customer-create-input))




### Type

#### [`Customer`](/developers/demandhq-apis/demand-api/graphql/reference/objects/customer) 

A customer of a given Storefront is a user who places orders on the Storefront
