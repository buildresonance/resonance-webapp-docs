Create a new Nav.
                
Requires the "nav/write" permission on the associated DemandHq.


```graphql
createNav(
  input: NavCreateInput!
): Nav

```


### Arguments

#### `input` ([`NavCreateInput`](/developers/demandhq-apis/demand-api/graphql/reference/inputs/nav-create-input))




### Type

#### [`Nav`](/developers/demandhq-apis/demand-api/graphql/reference/objects/nav) 

Nav is an item within a navigational heirarchy of some kind, most often viewed as a menu.
