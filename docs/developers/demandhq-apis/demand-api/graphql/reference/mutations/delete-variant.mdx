Delete a variant
            
Requires variant/write permission on the associated demandHq for the requesting identity


```graphql
deleteVariant(
  variantId: ID
): VariantDeleteResponse!

```


### Arguments

#### `variantId` ([`ID`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/id))




### Type

#### [`VariantDeleteResponse`](/developers/demandhq-apis/demand-api/graphql/reference/objects/variant-delete-response) 

Response for when deleting a variant
