Request access to sell items from a Supplier.
            
Requires the requesting identity to have the "demandhqsupplier/write" permission on the associated DemandHq.

```graphql
requestSupplierAccess(
  demandHqId: ID!
  supplierId: ID!
): DemandHqSupplier!
```

#### [`demandHqId`](#) ([`ID!`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/id))




#### [`supplierId`](#) ([`ID!`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/id))




### Type

#### [`DemandHqSupplier`](/developers/demandhq-apis/demand-api/graphql/reference/objects/demand-hq-supplier)

A relationship between a Supplier and a DemandHQ.
