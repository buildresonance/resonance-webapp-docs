Updates an existing Customer

Requires the customer/write scope on the associated DemandHq.

```graphql
updateCustomer(
  input: CustomerUpdateInput!
): Customer
```

#### [`input`](#) ([`CustomerUpdateInput!`](/developers/demandhq-apis/demand-api/graphql/reference/inputs/customer-update-input))




### Type

#### [`Customer`](/developers/demandhq-apis/demand-api/graphql/reference/objects/customer)
