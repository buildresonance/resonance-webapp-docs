Update an existing PageTypeContent.

Requires the pagetypecontent/write scope on the associated DemandHq.


```graphql
updatePageTypeContent(
  input: PageTypeContentUpdateInput!
): PageTypeContent!

```


### Arguments

#### `input` ([`PageTypeContentUpdateInput`](/developers/demandhq-apis/demand-api/graphql/reference/inputs/page-type-content-update-input))




### Type

#### [`PageTypeContent`](/developers/demandhq-apis/demand-api/graphql/reference/objects/page-type-content) 

This holds content specific to a pageType. This is generally used on a page that lists all pages for a pageType.
