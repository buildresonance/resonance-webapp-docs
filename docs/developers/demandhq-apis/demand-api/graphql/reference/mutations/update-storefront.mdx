Update an existing Storefront.
    
Requires the "storefront/write" scope on the associated DemandHq.


```graphql
updateStorefront(
  input: StorefrontUpdateInput
): Storefront

```


### Arguments

#### `input` ([`StorefrontUpdateInput`](/developers/demandhq-apis/demand-api/graphql/reference/inputs/storefront-update-input))




### Type

#### [`Storefront`](/developers/demandhq-apis/demand-api/graphql/reference/objects/storefront) 

A Storefront represents a customer facing ecommerce website that consumes Resonance Demand Entities
