Add or Update CollectionContent for an existing Collection.

Requires the collection/write permission on the given DemandHq


```graphql
upsertCollectionContent(
  input: CollectionContentUpsertInput!
): CollectionContent!

```


### Arguments

#### `input` ([`CollectionContentUpsertInput`](/developers/demandhq-apis/demand-api/graphql/reference/inputs/collection-content-upsert-input))




### Type

#### [`CollectionContent`](/developers/demandhq-apis/demand-api/graphql/reference/objects/collection-content) 

This is the actual markdown content for the collection
