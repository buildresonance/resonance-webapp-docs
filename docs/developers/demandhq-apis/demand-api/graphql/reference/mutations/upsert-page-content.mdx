Add or Update PageContent for an existing Page.

Requires the page/write permission on the given DemandHq


```graphql
upsertPageContent(
  input: PageContentUpsertInput!
): PageContent!

```


### Arguments

#### `input` ([`PageContentUpsertInput`](/developers/demandhq-apis/demand-api/graphql/reference/inputs/page-content-upsert-input))




### Type

#### [`PageContent`](/developers/demandhq-apis/demand-api/graphql/reference/objects/page-content) 

This is the actual content of the page
