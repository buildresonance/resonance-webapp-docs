A rule to determine whether a Product should be part of a Collection.


```graphql
type CollectionRule {
  field: CollectionRuleField!
  operator: CollectionRuleOperator!
  value: String!
}
```


### Fields

#### `field` ([`CollectionRuleField`](/developers/demandhq-apis/demand-api/graphql/reference/enums/collection-rule-field))

The field to match against


#### `operator` ([`CollectionRuleOperator`](/developers/demandhq-apis/demand-api/graphql/reference/enums/collection-rule-operator))

The operator to use when matching


#### `value` ([`String`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/string))

The value to match against
