A Product is a part of the Demand Domain and represents a the basic information of a product.
Products have many support entities, including variants, which represent SKUs of the product.


```graphql
type Product {
  createdAt: Date!
  demandHqId: ID!
  locale(
  localeType: ResonanceLocaleType!
): ProductLocale
  optionLabels: [String!]!
  productId: ID!
  publishedAt: Date
  seoTitle: String
  slug: String!
  status: ProductStatus!
  storefrontIds: [ID]
  tags: ProductTags
  title: String!
  updatedAt: Date!
  updatedBy: ID!
  variant(
  variantId: ID
): Variant
  variants(
  after: String
  before: String
  first: Int
  last: Int
): VariantConnection!
  vendor: String!
}
```


### Fields

#### `createdAt` ([`Date`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/date))




#### `demandHqId` ([`ID`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/id))




#### `locale` ([`ProductLocale`](/developers/demandhq-apis/demand-api/graphql/reference/objects/product-locale))

Get a single productLocale by ResonanceLocaleType

    Requires productlocale/read permission on the requesting identity
- ##### `localeType` ([`ResonanceLocaleType`](/developers/demandhq-apis/demand-api/graphql/reference/enums/resonance-locale-type))




#### `optionLabels` ([`String`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/string))

The labels for the possible option groups for this product, i.e. "Color", "Size", "Material", etc.
These will match with labelOptions in Variants.


#### `productId` ([`ID`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/id))




#### `publishedAt` ([`Date`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/date))

ISO format date when the product was published. Note that ensuing updates with
status set to PUBLISHED will republish and re-set this date.


#### `seoTitle` ([`String`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/string))

Title specific for SEO purposes, limit 256 characters.


#### `slug` ([`String`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/string))

Slug for the URL for this product. Limit 128 characters.


#### `status` ([`ProductStatus`](/developers/demandhq-apis/demand-api/graphql/reference/enums/product-status))

The status of the product, can be DRAFT or PUBLISHED.


#### `storefrontIds` ([`ID`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/id))

Which storefronts should have access to this product?


#### `tags` ([`ProductTags`](/developers/demandhq-apis/demand-api/graphql/reference/objects/product-tags))

Get the tags for this product


#### `title` ([`String`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/string))

The title of the product, limit 256 characters.


#### `updatedAt` ([`Date`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/date))




#### `updatedBy` ([`ID`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/id))




#### `variant` ([`Variant`](/developers/demandhq-apis/demand-api/graphql/reference/objects/variant))

Get a single variant by VariantId

    Requires demandHqvariant/read permission on the requesting identity
- ##### `variantId` ([`ID`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/id))




#### `variants` ([`VariantConnection`](/developers/demandhq-apis/demand-api/graphql/reference/objects/variant-connection))

Get a list of variants

    Requires demandHqvariant/read permission on the demandHq for the requesting identity
- ##### `after` ([`String`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/string))

- ##### `before` ([`String`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/string))

- ##### `first` ([`Int`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/int))

Returns the first n elements from the list.


- ##### `last` ([`Int`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/int))

#### `vendor` ([`String`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/string))

AKA Brand. Limit 256 characters.
