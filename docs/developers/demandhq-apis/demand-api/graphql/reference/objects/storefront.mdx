A Storefront represents a customer facing ecommerce website that consumes Resonance Demand Entities


```graphql
type Storefront {
  apiKey(
  slug: String!
): StorefrontApiKey
  apiKeys: [StorefrontApiKey]!
  createdAt: Date!
  demandHqId: ID!
  description: String
  name: String!
  storefrontId: ID!
  storefrontIdentities: [StorefrontIdentity!]
  updatedAt: Date!
  updatedBy: ID!
  urlDomain: String!
}
```
