Get a Customer by their customerId.
            
Requires the customer/read permission for the associated DemandHq.


```graphql
customer(
  customerId: ID!
): Customer

```


### Arguments

#### `customerId` ([`ID`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/id))




### Type

#### [`Customer`](/developers/demandhq-apis/demand-api/graphql/reference/objects/customer) 

A customer of a given Storefront is a user who places orders on the Storefront
