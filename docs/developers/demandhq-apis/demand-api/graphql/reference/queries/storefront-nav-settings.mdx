Retrieve the settings for a storefront's Navs
            
Requires the storefrontnavsettings/read permission on the associated DemandHq.

```graphql
storefrontNavSettings(
  storefrontId: ID!
): StorefrontNavSettings

```


### Arguments

#### `storefrontId` ([`ID`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/id))




### Type

#### [`StorefrontNavSettings`](/developers/demandhq-apis/demand-api/graphql/reference/objects/storefront-nav-settings) 

The settings for a storefront's Navs
