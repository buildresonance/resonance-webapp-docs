Get a storefront by its storefrontId.
            
Requires the storefront/read permission on the associated DemandHq.


```graphql
storefront(
  storefrontId: ID!
): Storefront

```


### Arguments

#### `storefrontId` ([`ID`](/developers/demandhq-apis/demand-api/graphql/reference/scalars/id))




### Type

#### [`Storefront`](/developers/demandhq-apis/demand-api/graphql/reference/objects/storefront) 

A Storefront represents a customer facing ecommerce website that consumes Resonance Demand Entities
