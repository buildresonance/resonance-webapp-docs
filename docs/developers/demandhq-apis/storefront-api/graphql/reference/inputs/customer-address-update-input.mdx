Input for updating a CustomerAddress


```graphql
input CustomerAddressUpdateInput {
  city: String
  company: String
  country: String
  customerAddressId: ID!
  name: String
  phoneNumber: String
  postalCode: String
  state: String
  streetAddress1: String
  streetAddress2: String
}
```


### Fields

#### `city` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The city of the address


#### `company` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The company associated with the address


#### `country` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The country of the address


#### `customerAddressId` ([`ID`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/id))




#### `name` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The name associated with the address, i.e. the Person in the household


#### `phoneNumber` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The phone number of the address. Format +11231231234


#### `postalCode` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The zip or postal code of the address


#### `state` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The state of the address


#### `streetAddress1` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The first line of the street address


#### `streetAddress2` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The second line of the street address
