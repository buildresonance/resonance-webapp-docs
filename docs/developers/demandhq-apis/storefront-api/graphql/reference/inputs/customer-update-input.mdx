Input for updating a Customer


```graphql
input CustomerUpdateInput {
  customerId: ID!
  defaultBillingAddressId: ID
  defaultShippingAddressId: ID
  emailAddress: String
  emailMarketingConsent: CustomerMarketingConsentInput
  firstName: String
  lastName: String
  phoneNumber: String
  smsMarketingConsent: CustomerMarketingConsentInput
}
```


### Fields

#### `customerId` ([`ID`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/id))




#### `defaultBillingAddressId` ([`ID`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/id))

The CustomerAddressId for the default billing address for this Customer


#### `defaultShippingAddressId` ([`ID`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/id))

The CustomerAddressId for the default shipping address for this Customer


#### `emailAddress` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The email address associated with the Customer, must be unique to the storefront, limit 256 characters


#### `emailMarketingConsent` ([`CustomerMarketingConsentInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/customer-marketing-consent-input))




#### `firstName` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The first name of the Customer, limit 256 characters


#### `lastName` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

The last name of the Customer, limit 256 characters


#### `phoneNumber` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Must be unique to the Storefront. Limit 64 characters. Format +11231231234


#### `smsMarketingConsent` ([`CustomerMarketingConsentInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/customer-marketing-consent-input))
