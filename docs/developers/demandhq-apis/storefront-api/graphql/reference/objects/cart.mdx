A cart is a temporary container for products that a customer or anonymous user wants to buy


```graphql
type Cart {
  cartItems: [CartItem]
  cartType: CartType!
  demandHqId: ID!
  externalCustomerOrDeviceId: String!
  savedForLaterItems: [SavedForLaterItem]
  storefrontId: ID!
  updatedAt: Date!
}
```
