These are the tags associated to the collection


```graphql
type CollectionTags {
  parsed(
  category: String
  key: String
): [ParsedTag]
  raw: [String]
}
```


### Fields

#### `parsed` ([`ParsedTag`](/developers/demandhq-apis/storefront-api/graphql/reference/objects/parsed-tag))

Get parsed, and optionally filtered, tags for a Collection.
- ##### `category` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Optionally filter tags by category.


- ##### `key` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Optionally filter tags by key. If category is not provided, but key is, category must be null to be returned.
