Paginate through PublicProducts

```graphql
type PublicProductConnectionWithSearch {
  edges: [PublicProductEdge]!
  nodes: [PublicProduct]!
  pageInfo: PageInfo!
  querySuggestions: JSON
  termAggregations: ProductTermAggregations
}
```

#### [`nodes`](#) ([`[PublicProduct]!`](/developers/demandhq-apis/storefront-api/graphql/reference/objects/public-product))




#### [`pageInfo`](#) ([`PageInfo!`](/developers/demandhq-apis/storefront-api/graphql/reference/objects/page-info))




#### [`querySuggestions`](#) ([`JSON`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/json))

This is an object with terms in the query as keys, and a suggestion for that
term as the value. This is often referred to as 'Did you mean?'


#### [`termAggregations`](#) ([`ProductTermAggregations`](/developers/demandhq-apis/storefront-api/graphql/reference/objects/product-term-aggregations))

Values and the number of times they occur for the results of the query. These
will be the same for every page of a query, and should only be fetched on the first page.
