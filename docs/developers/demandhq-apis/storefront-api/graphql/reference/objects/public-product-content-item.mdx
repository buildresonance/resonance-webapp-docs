This has a many to one relationship with Product, and provides a connection to the CDN items.

```graphql
type PublicProductContentItem {
  alt: String!
  category: String
  contentItemId: ID!
  contentKey: String!
  contentType: ProductContentType!
  demandHqId: ID!
  position: Int
  productId: ID!
}
```
