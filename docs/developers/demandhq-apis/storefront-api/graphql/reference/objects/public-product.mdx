A Product is a part of the Demand Domain and represents a the basic information of a product.
Products have many support entities, including variants, which represent SKUs of the product.

```graphql
type PublicProduct {
  demandHqId: ID!
  locale(
  localeType: ResonanceLocaleType!
): PublicProductLocale
  optionLabels: [String!]!
  productId: ID!
  publishedAt: Date
  seoTitle: String
  slug: String!
  tags: ProductTags
  title: String!
  variant(
  variantId: ID
): PublicVariant
  variants(
  after: String
  before: String
  first: Int
  last: Int
): PublicVariantConnection!
  vendor: String!
}
```
