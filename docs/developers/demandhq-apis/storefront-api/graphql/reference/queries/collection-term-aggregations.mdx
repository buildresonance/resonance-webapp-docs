Term aggregations for Collections, typically used to suggest terms to use in Search Filters.

```graphql
collectionTermAggregations(
  numberOfResults: Int!
): CollectionTermAggregations
```


### Arguments

#### [`numberOfResults`](#) ([`Int!`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/int))

The number of aggregations to return for each type.


### Type

#### [`CollectionTermAggregations`](/developers/demandhq-apis/storefront-api/graphql/reference/objects/collection-term-aggregations)

A Map of several fields of Collection and term aggregations for each
