Get a collectionTypeContent by collectionType.
                
Requires the collectiontypecontent/read scope on the associated DemandHq

```graphql
collectionTypeContent(
  collectionType: String!
): PublicCollectionTypeContent
```


### Arguments

#### [`collectionType`](#) ([`String!`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))




### Type

#### [`PublicCollectionTypeContent`](/developers/demandhq-apis/storefront-api/graphql/reference/objects/public-collection-type-content)

This holds content specific to a collectionType. This is generally used on a
page that lists all collections for a collectionType.
