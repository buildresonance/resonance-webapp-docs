List collections for a Storefront with optional filters.
            
Requires the publiccollection/read scope.

```graphql
collections(
  after: String
  before: String
  collectionType: StringFilterInput
  first: Int
  last: Int
  query: String
  slugs: [String]
  sortOrder: [CollectionSortInput]
  tags: [StringFilterInput]
): PublicCollectionConnectionWithSearch!
```


### Arguments

#### [`after`](#) ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Returns the elements in the list that come after the specified cursor.


#### [`before`](#) ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Returns the elements in the list that come before the specified cursor.


#### [`collectionType`](#) ([`StringFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/string-filter-input))

Filter Collections using collectionType


#### [`first`](#) ([`Int`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/int))

Returns the first n elements from the list.


#### [`last`](#) ([`Int`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/int))

Returns the last n elements from the list.


#### [`query`](#) ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Query collections for certain terms. Searches several fields, including
collection.description, collection.subtitle, collection.title, and
collectionTags.


#### [`slugs`](#) ([`[String]`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Filter Collections using slugs


#### [`sortOrder`](#) ([`[CollectionSortInput]`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/collection-sort-input))

Sort Collections by a series of fields and directions.


#### [`tags`](#) ([`[StringFilterInput]`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/string-filter-input))

Filter Collections using Tags.


### Type

#### [`PublicCollectionConnectionWithSearch`](/developers/demandhq-apis/storefront-api/graphql/reference/objects/public-collection-connection-with-search)

Paginate through PublicCollections
