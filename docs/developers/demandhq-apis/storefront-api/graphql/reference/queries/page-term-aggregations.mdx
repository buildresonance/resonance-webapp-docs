Term aggregations for Pages, typically used to suggest terms to use in Search Filters.


```graphql
pageTermAggregations(
  numberOfResults: Int!
): PageTermAggregations

```


### Arguments

#### `numberOfResults` ([`Int`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/int))

The number of aggregations to return for each type.


### Type

#### [`PageTermAggregations`](/developers/demandhq-apis/storefront-api/graphql/reference/objects/page-term-aggregations) 

A Map of several fields of Page and term aggregations for each
