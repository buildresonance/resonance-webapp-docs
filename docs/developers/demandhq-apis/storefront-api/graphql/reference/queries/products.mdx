Paginate through products.

Requires the publicproduct/read scope.


```graphql
products(
  after: String
  before: String
  first: Int
  last: Int
  manufacturer: [StringFilterInput]
  msrp: FloatRangeFilterInput
  price: FloatRangeFilterInput
  query: String
  slugs: [String]
  sortOrder: [ProductSortInput]
  standardizedColor: [ProductStandardizedColorFilterInput]
  tags: [StringFilterInput]
  vendor: StringFilterInput
): PublicProductConnectionWithSearch!

```


### Arguments

#### `after` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Returns the elements in the list that come after the specified cursor.


#### `before` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Returns the elements in the list that come before the specified cursor.


#### `first` ([`Int`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/int))

Returns the first n elements from the list.


#### `last` ([`Int`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/int))

Returns the last n elements from the list.


#### `manufacturer` ([`StringFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/string-filter-input))

Optionally filter by manufacturer.


#### `msrp` ([`FloatRangeFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/float-range-filter-input))

Optionally filter by msrp.


#### `price` ([`FloatRangeFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/float-range-filter-input))

Optionally filter by price.


#### `query` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Query products for certain terms. Searches several fields, including
product.tags, product.title, product.vendor, productLocales.description,
productLocales.featureBullets, variant.barcode, variant.mpn, variant.sku


#### `slugs` ([`String`](/developers/demandhq-apis/storefront-api/graphql/reference/scalars/string))

Filter Products using slugs


#### `sortOrder` ([`ProductSortInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/product-sort-input))

Sort Products by a series of fields and directions, defaults to CREATED_AT/ASC.


#### `standardizedColor` ([`ProductStandardizedColorFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/product-standardized-color-filter-input))

Optionally filter by standardized color.


#### `tags` ([`StringFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/string-filter-input))

Optionally filter by tags.


#### `vendor` ([`StringFilterInput`](/developers/demandhq-apis/storefront-api/graphql/reference/inputs/string-filter-input))

Optionally filter by vendor.


### Type

#### [`PublicProductConnectionWithSearch`](/developers/demandhq-apis/storefront-api/graphql/reference/objects/public-product-connection-with-search) 

Paginate through PublicProducts
