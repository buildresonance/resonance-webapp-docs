The possible fields in order to update an existing app


```graphql
input AppUpdateInput {
  appId: ID!
  description: String
  name: String
  requestedScopes: [String]
  status: AppStatus
}
```


### Fields

#### `appId` ([`ID`](/developers/shared-apis/adminidentity-api/graphql/reference/scalars/id))




#### `description` ([`String`](/developers/shared-apis/adminidentity-api/graphql/reference/scalars/string))

Short description of the app
Limited to 8000 characters


#### `name` ([`String`](/developers/shared-apis/adminidentity-api/graphql/reference/scalars/string))

The name of the app
Limited to 128 characters


#### `requestedScopes` ([`String`](/developers/shared-apis/adminidentity-api/graphql/reference/scalars/string))




#### `status` ([`AppStatus`](/developers/shared-apis/adminidentity-api/graphql/reference/enums/app-status))
