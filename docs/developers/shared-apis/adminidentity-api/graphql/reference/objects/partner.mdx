Represents a Partner, the parent to an App, and the parent most component for Development

```graphql
type Partner {
  app(appId: ID!): App
  apps(after: String, before: String, first: Int, last: Int): AppConnection!
  createdAt: String!
  description: String
  name: String!
  notificationUserNames: PartnerNotificationUserNames
  ownerUserName: String!
  partnerId: ID!
  partnerIdentities(
    after: String
    before: String
    first: Int
    last: Int
  ): PartnerIdentityConnection!
  partnerIdentity(identityId: ID!): PartnerIdentity
  updatedAt: String!
}
```
