Represents a Partner, the parent to an App, and the parent most component for Development

```graphql
type Partner {
  app(appId: ID!): App
  apps(
    after: String
    before: String
    first: Int = 0
    last: Int = 0
  ): AppConnection!
  createdAt: String!
  description: String
  name: String!
  ownerUserName: String!
  partnerId: ID!
  partnerUserPermission(userName: ID!): PartnerUserPermission
  partnerUserPermissions(
    after: String
    before: String
    first: Int = 0
    last: Int = 0
  ): PartnerUserPermissionConnection!
  updatedAt: String!
}
```
