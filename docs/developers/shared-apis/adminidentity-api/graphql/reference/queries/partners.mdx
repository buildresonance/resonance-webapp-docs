Get a list of partners

```graphql
partners(
  first: Int = 0
  last: Int = 0
  before: String
  after: String
): PartnerConnection!

```

### Arguments

#### `first` ([`Int`](/developers/shared-apis/adminidentity-api/graphql/reference/scalars/int))

Returns the first n elements from the list.

#### `last` ([`Int`](/developers/shared-apis/adminidentity-api/graphql/reference/scalars/int))

Returns the last n elements from the list.

#### `before` ([`String`](/developers/shared-apis/adminidentity-api/graphql/reference/scalars/string))

Returns the elements in the list that come before the specified cursor.

#### `after` ([`String`](/developers/shared-apis/adminidentity-api/graphql/reference/scalars/string))

Returns the elements in the list that come after the specified cursor.

### Type

#### [`PartnerConnection`](/developers/shared-apis/adminidentity-api/graphql/reference/objects/partner-connection)
