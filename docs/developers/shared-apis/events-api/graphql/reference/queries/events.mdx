Get a list of events

```graphql
events(
  domainEntityId: String!
  entityId: String!
  first: Int = 0
  last: Int = 0
  before: String
  after: String
): ResonanceEventConnection!

```

### Arguments

#### `domainEntityId` ([`String!`](/developers/shared-apis/events-api/graphql/reference/scalars/string))

This is most typically domainHqId or supplierId

#### `entityId` ([`String!`](/developers/shared-apis/events-api/graphql/reference/scalars/string))

This is the id of the item the event occurred for, i.e. productId

#### `first` ([`Int`](/developers/shared-apis/events-api/graphql/reference/scalars/int))

Returns the first n elements from the list.

#### `last` ([`Int`](/developers/shared-apis/events-api/graphql/reference/scalars/int))

Returns the last n elements from the list.

#### `before` ([`String`](/developers/shared-apis/events-api/graphql/reference/scalars/string))

Returns the elements in the list that come before the specified cursor.

#### `after` ([`String`](/developers/shared-apis/events-api/graphql/reference/scalars/string))

Returns the elements in the list that come after the specified cursor.

### Type

#### [`ResonanceEventConnection`](/developers/shared-apis/events-api/graphql/reference/objects/resonance-event-connection)

Lists of Resonance Events
