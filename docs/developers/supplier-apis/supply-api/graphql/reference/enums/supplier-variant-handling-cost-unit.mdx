Indicates what unit the amount is in

```graphql
enum SupplierVariantHandlingCostUnit {
  CURRENCY
  PERCENTAGE_OF_COST
}
```

### Values

#### `CURRENCY`

The amount is in a currency, i.e. $

#### `PERCENTAGE_OF_COST`

The amount is a percentage of the Cost field
