Input object for creating an Image SupplierProductContentItem from a File on a device

```graphql
input SupplierProductImageFromFileInput {
  alt: String!
  category: String
  fileExtension: String!
  fileName: String!
  position: Int
  supplierId: ID!
  supplierProductId: ID!
}
```
