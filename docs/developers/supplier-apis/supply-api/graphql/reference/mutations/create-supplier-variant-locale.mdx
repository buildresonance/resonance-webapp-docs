Create a new supplierVariantLocale

```graphql
createSupplierVariantLocale(
  input: SupplierVariantLocaleCreateInput!
): SupplierVariantLocale

```

### Arguments

#### `input` ([`SupplierVariantLocaleCreateInput!`](/developers/supplier-apis/supply-api/graphql/reference/inputs/supplier-variant-locale-create-input))

### Type

#### [`SupplierVariantLocale`](/developers/supplier-apis/supply-api/graphql/reference/objects/supplier-variant-locale)

This has a many to one relationship with SupplierVariant, and provides localized language data for the appropriate fields.
There will often only be one of these, and it should be the one set in defaultLocale for the supplier.
