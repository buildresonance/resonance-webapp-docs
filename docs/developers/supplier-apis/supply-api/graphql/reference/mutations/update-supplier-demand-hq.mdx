Updates an existing SupplierDemandHq.

            Requires the requesting identity to have the "supplierdemandhq/write" permission on the associated Supplier.

```graphql
updateSupplierDemandHq(
  input: SupplierDemandHqUpdateInput!
): SupplierDemandHq!
```

#### [`input`](#) ([`SupplierDemandHqUpdateInput!`](/developers/supplier-apis/supply-api/graphql/reference/inputs/supplier-demand-hq-update-input))




### Type

#### [`SupplierDemandHq`](/developers/supplier-apis/supply-api/graphql/reference/objects/supplier-demand-hq)

A relationship between a Supplier and a DemandHQ.
