Update a SupplierIdentity for a Supplier (give a user permissions)

```graphql
updateSupplierIdentity(
  input: SupplierIdentityUpdateInput!
): SupplierIdentity

```

### Arguments

#### `input` ([`SupplierIdentityUpdateInput`](/developers/supplier-apis/supply-api/graphql/reference/inputs/supplier-identity-update-input))

### Type

#### [`SupplierIdentity`](/developers/supplier-apis/supply-api/graphql/reference/objects/supplier-identity)

Indicates a User or App has permissions on a Supplier
