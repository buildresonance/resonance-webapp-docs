Update an existing supplierVariant

```graphql
updateSupplierVariant(

): SupplierVariant!

```


### Arguments

#### `input` ([`SupplierVariantUpdateInput`](/developers/supplier-apis/supply-api/graphql/reference/inputs/supplier-variant-update-input))




### Type

#### [`SupplierVariant`](/developers/supplier-apis/supply-api/graphql/reference/objects/supplier-variant) 

The main item of the Supplier Domain, represents a single SKU of a product
Only the variant level exists, there is no parent product.
