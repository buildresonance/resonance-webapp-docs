The globally used Address of Resonance


```graphql
type Address {
  city: String!
  country: String!
  phoneNumber: String
  postalCode: String!
  state: String!
  streetAddress1: String!
  streetAddress2: String
}
```


### Fields

#### `city` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

The city of the address


#### `country` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

The country of the address


#### `phoneNumber` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

The phone number of the address. Format +11231231234


#### `postalCode` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

The zip or postal code of the address


#### `state` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

The state of the address


#### `streetAddress1` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

The first line of the street address


#### `streetAddress2` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

The second line of the street address
