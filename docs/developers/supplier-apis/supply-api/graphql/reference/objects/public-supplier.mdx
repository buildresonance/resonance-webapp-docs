The parent-most entity of the Supplier Domain


```graphql
type PublicSupplier {
  category: SupplierCategory
  description: String
  inStockProducts: Int
  name: String!
  status: SupplierStatus!
  supplierId: ID!
  topVendors: [String!]
}
```


### Fields

#### `category` ([`SupplierCategory`](/developers/supplier-apis/supply-api/graphql/reference/enums/supplier-category))

Indicates the vertical the majority of a Supplier's goods are in.


#### `description` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))




#### `inStockProducts` ([`Int`](/developers/supplier-apis/supply-api/graphql/reference/scalars/int))

The approximate number of in stock products this Supplier has on a typical basis.


#### `name` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))




#### `status` ([`SupplierStatus`](/developers/supplier-apis/supply-api/graphql/reference/enums/supplier-status))




#### `supplierId` ([`ID`](/developers/supplier-apis/supply-api/graphql/reference/scalars/id))




#### `topVendors` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

The top Brands or Vendors this Supplier carries, limit 5 and 256 characters each.
