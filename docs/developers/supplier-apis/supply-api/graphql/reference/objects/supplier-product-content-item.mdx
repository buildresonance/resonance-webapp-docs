This has a many to one relationship with SupplierProduct, and provides a connection to the CDN items.


```graphql
type SupplierProductContentItem {
  alt: String!
  category: String
  contentItemId: ID!
  contentKey: String!
  contentType: SupplierProductContentType!
  createdAt: Date!
  position: Int
  supplierId: ID!
  supplierProductId: ID!
  updatedAt: Date!
  updatedBy: String!
}
```
