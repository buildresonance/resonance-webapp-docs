Categories and other groupings that apply to this product.


```graphql
type SupplierProductTags {
  parsed(
  category: String
  key: String
): ParsedTag
  raw: [String!]
  supplierId: ID!
  supplierProductId: ID!
  updatedAt: Date!
  updatedBy: ID!
}
```


### Fields

#### `parsed` ([`ParsedTag`](/developers/supplier-apis/supply-api/graphql/reference/objects/parsed-tag))

Get parsed, and optionally filtered, tags for a SupplierProduct.
- ##### `category` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

Optionally filter tags by category.


- ##### `key` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

Optionally filter tags by key. If category is not provided, but key is, category must be null to be returned.
