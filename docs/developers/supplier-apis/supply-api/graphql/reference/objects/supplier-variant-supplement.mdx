This has a one to one relationship with SupplierVariant, and provides supplemental data for the SupplierVariant.
These items get updated at a lower frequency, and are therefore separate.


```graphql
type SupplierVariantSupplement {
  averageShippingCost: Float
  barcode: String!
  casePack: Int
  cost: Float!
  createdAt: String!
  handlingCost: SupplierVariantHandlingCost
  handlingCostAsCurrency: Float
  isTaxable: Boolean
  manufacturer: String
  mapPrice: Float
  mpn: String
  msrp: Float
  productTags: [String]
  recommendedPrice: Float
  shippingDimensions: SupplierVariantShippingDimensions
  sku: String
  standardizedColor: SupplierVariantStandardizedColor
  supplierId: ID!
  supplierProductId: ID!
  supplierVariantId: ID!
  updatedAt: String!
  updatedBy: String!
  weight: SupplierVariantWeight
}
```


### Fields

#### `averageShippingCost` ([`Float`](/developers/supplier-apis/supply-api/graphql/reference/scalars/float))

A calculated metric for some kind of average for a common region. Uncommon.


#### `barcode` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

UPC, ISBN, GTIN, etc


#### `casePack` ([`Int`](/developers/supplier-apis/supply-api/graphql/reference/scalars/int))

Quantity in one variant (i.e. 12-pack)


#### `cost` ([`Float`](/developers/supplier-apis/supply-api/graphql/reference/scalars/float))

COGS, or Cost of Goods Sold, in a currency


#### `createdAt` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

ISO Date


#### `handlingCost` ([`SupplierVariantHandlingCost`](/developers/supplier-apis/supply-api/graphql/reference/objects/supplier-variant-handling-cost))

Additional warehouse handling charge, typically for larger items.


#### `handlingCostAsCurrency` ([`Float`](/developers/supplier-apis/supply-api/graphql/reference/scalars/float))

Calculated field, giving the handling cost as Currency no matter how it is stored

#### `mapPrice` ([`Float`](/developers/supplier-apis/supply-api/graphql/reference/scalars/float))

#### `mpn` ([`String`](/developers/supplier-apis/supply-api/graphql/reference/scalars/string))

#### `msrp` ([`Float`](/developers/supplier-apis/supply-api/graphql/reference/scalars/float))
