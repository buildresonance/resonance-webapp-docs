Status of the PurchaseOrderItem
The status is a read-only calculated by the API.

```graphql
enum PurchaseOrderItemStatus {
  CANCELLED
  DRAFT
  FULFILLED
  INITIAL
  PARTIALLY_FULFILLED
  PROCESSING
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>PurchaseOrderItemStatus.<b>CANCELLED</b></code>](#) 
> Indicates that the PurchaseOrderItem was placed, but is no longer needed to be fulfilled.
> 

#### [<code style={{ fontWeight: 'normal' }}>PurchaseOrderItemStatus.<b>DRAFT</b></code>](#) 
> Indicates that the PurchaseOrderItem is currently in the process of being created, and data is incomplete.
> 

#### [<code style={{ fontWeight: 'normal' }}>PurchaseOrderItemStatus.<b>FULFILLED</b></code>](#) 
> Indicates that the PurchaseOrderItem is fully fulfilled and is now complete.
> 

#### [<code style={{ fontWeight: 'normal' }}>PurchaseOrderItemStatus.<b>INITIAL</b></code>](#) 
> Indicates that the PurchaseOrderItem is ready for processing, and data is complete.
> 

#### [<code style={{ fontWeight: 'normal' }}>PurchaseOrderItemStatus.<b>PARTIALLY_FULFILLED</b></code>](#) 
> Indicates that the PurchaseOrderItem is in the process of being fulfilled.
> 

#### [<code style={{ fontWeight: 'normal' }}>PurchaseOrderItemStatus.<b>PROCESSING</b></code>](#) 
> Indicates that the PurchaseOrderItem now just beginning to be fulfilled. This
also means that the item can no longer be cancelled.
>
