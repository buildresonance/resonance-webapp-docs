The input type for an Address

```graphql
input AddressInput {
  city: String!
  company: String
  country: String!
  name: String!
  phoneNumber: String
  postalCode: String!
  state: String!
  streetAddress1: String!
  streetAddress2: String
}
```


### Fields

#### [`city`](#) ([`String!`](/supply/api/graphql-admin/reference/scalars/string))

The city of the address


#### [`company`](#) ([`String`](/supply/api/graphql-admin/reference/scalars/string))

The company associated with the address


#### [`country`](#) ([`String!`](/supply/api/graphql-admin/reference/scalars/string))

The country of the address


#### [`name`](#) ([`String!`](/supply/api/graphql-admin/reference/scalars/string))

The name associated with the address, i.e. the Person in the household


#### [`phoneNumber`](#) ([`String`](/supply/api/graphql-admin/reference/scalars/string))

The phone number of the address. Format +11231231234


#### [`postalCode`](#) ([`String!`](/supply/api/graphql-admin/reference/scalars/string))

The zip or postal code of the address


#### [`state`](#) ([`String!`](/supply/api/graphql-admin/reference/scalars/string))

The state of the address


#### [`streetAddress1`](#) ([`String!`](/supply/api/graphql-admin/reference/scalars/string))

The first line of the street address


#### [`streetAddress2`](#) ([`String`](/supply/api/graphql-admin/reference/scalars/string))

The second line of the street address
