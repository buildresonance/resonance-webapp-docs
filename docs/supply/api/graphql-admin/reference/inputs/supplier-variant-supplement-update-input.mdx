The necessary fields in order to update a supplierVariantSupplement.
ID fields required, all other fields optional, only included fields will be updated.

```graphql
input SupplierVariantSupplementUpdateInput {
  averageShippingCost: Float
  barcode: String
  casePack: Int
  cost: Float
  handlingCost: SupplierVariantHandlingCostInput
  isTaxable: Boolean
  manufacturer: String
  mapPrice: Float
  mpn: String
  msrp: Float
  recommendedPrice: Float
  shippingDimensions: SupplierVariantShippingDimensionsInput
  sku: String
  standardizedColor: VariantStandardizedColor
  supplierVariantId: ID!
  weight: SupplierVariantWeightInput
}
```


### Fields

#### [`averageShippingCost`](#) ([`Float`](/supply/api/graphql-admin/reference/scalars/float))

A calculated metric for some kind of average for a common region. Uncommon.


#### [`barcode`](#) ([`String`](/supply/api/graphql-admin/reference/scalars/string))

UPC, ISBN, GTIN, etc


#### [`casePack`](#) ([`Int`](/supply/api/graphql-admin/reference/scalars/int))

Quantity in one variant (i.e. 12-pack)


#### [`cost`](#) ([`Float`](/supply/api/graphql-admin/reference/scalars/float))

COGS, or Cost of Goods Sold, in a currency


#### [`handlingCost`](#) ([`SupplierVariantHandlingCostInput`](/supply/api/graphql-admin/reference/inputs/supplier-variant-handling-cost-input))

Additional warehouse handling charge, typically for larger items.

Minimum Advertised Price


#### [`mpn`](#) ([`String`](/supply/api/graphql-admin/reference/scalars/string))

Manufacturer Part Number


#### [`msrp`](#) ([`Float`](/supply/api/graphql-admin/reference/scalars/float))

Manufacturer's Suggested Retail Price
