Creates a new SupplierDemandHq.
            
Requires the requesting identity to have the "supplierdemandhq/write" permission on the associated Supplier.

```graphql
createSupplierDemandHq(
  input: SupplierDemandHqCreateInput!
): SupplierDemandHq!
```


### Arguments

#### [`input`](#) ([`SupplierDemandHqCreateInput!`](/supply/api/graphql-admin/reference/inputs/supplier-demand-hq-create-input))




### Type

#### [`SupplierDemandHq`](/supply/api/graphql-admin/reference/objects/supplier-demand-hq)

A relationship between a Supplier and a DemandHQ.
