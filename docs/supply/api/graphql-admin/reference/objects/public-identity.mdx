This type is returned on routes with no permissions attached to protect
Personally Identifiable Information, but still provide minimal data.
Can be a User or AppClient.

```graphql
type PublicIdentity {
  displayName: String!
  identityId: ID!
  identityParentEntityId: ID
}
```
