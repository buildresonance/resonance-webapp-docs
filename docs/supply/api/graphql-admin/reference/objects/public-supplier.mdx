The parent-most entity of the Supplier Domain

```graphql
type PublicSupplier {
  category: SupplierCategory
  description: String
  inStockProducts: Int
  name: String!
  shipFromAddress: Address
  status: SupplierStatus!
  supplierId: ID!
  topVendors: [String!]
}
```
