Contains before and after snapshots for every event that occurs within the Supply domain

```graphql
type SupplyEvent {
  message: SupplyEventMessage
  messageId: String!
  subject: String
  timestamp: Date!
}
```


### Fields

#### [`message`](#) ([`SupplyEventMessage`](/supply/api/graphql-admin/reference/objects/supply-event-message))

This is the event data and metadata.
The data is variable, but most typically contains a newItem and oldItem.


#### [`messageId`](#) ([`String!`](/supply/api/graphql-admin/reference/scalars/string))

UUID


#### [`subject`](#) ([`String`](/supply/api/graphql-admin/reference/scalars/string))

A human-friendly string summarizing the event


#### [`timestamp`](#) ([`Date!`](/supply/api/graphql-admin/reference/scalars/date))

ISO Date that the event occurred
