Get a SupplierDemandHq by its Ids.
            
Requires the requesting identity to have the "supplierdemandhq/read" permission on the associated Supplier.

```graphql
supplierDemandHq(
  demandHqId: ID!
  supplierId: ID!
): SupplierDemandHq
```
