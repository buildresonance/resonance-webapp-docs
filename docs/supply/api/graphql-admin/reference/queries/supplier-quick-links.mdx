List multiple SupplierQuickLinks for an app.

At least one filter is required, supplierId and quickLinkType are required to be used together.

Permissions vary by filter:
- appId: You must be an appClient for the app.
- supplierId: You must have supplier/read permission for the supplier.

```graphql
supplierQuickLinks(
  after: String
  appId: ID
  before: String
  first: Int
  last: Int
  quickLinkType: SupplierQuickLinkType
  supplierId: ID
): SupplierQuickLinkConnection!
```


### Arguments

#### [`after`](#) ([`String`](/supply/api/graphql-admin/reference/scalars/string))

Returns the elements in the list that come after the specified cursor.


#### [`appId`](#) ([`ID`](/supply/api/graphql-admin/reference/scalars/id))

Filter for AppId, cannot be used with other filters.


#### [`before`](#) ([`String`](/supply/api/graphql-admin/reference/scalars/string))

Returns the elements in the list that come before the specified cursor.


#### [`first`](#) ([`Int`](/supply/api/graphql-admin/reference/scalars/int))

Returns the first n elements from the list.


#### [`last`](#) ([`Int`](/supply/api/graphql-admin/reference/scalars/int))

Returns the last n elements from the list.


#### [`quickLinkType`](#) ([`SupplierQuickLinkType`](/supply/api/graphql-admin/reference/enums/supplier-quick-link-type))

Filter to be used with supplierId, cannot be used with other filters.


#### [`supplierId`](#) ([`ID`](/supply/api/graphql-admin/reference/scalars/id))

Filter to be used with quickLinkType, cannot be used with other filters.


### Type

#### [`SupplierQuickLinkConnection`](/supply/api/graphql-admin/reference/objects/supplier-quick-link-connection)

Paginate through SupplierQuickLinks
